# 여행 트렌드 검색 분석, 예측 앱 사이트

## 1. 개요

### 1) 기회 의도
 - 과거 해외 및 국내 여행 검색 순위에 따라 트렌드를 확인할 수 있게 해주는 앱니다.
 - 과거의 데이터를 통해 이후 미래의 검색 트렌드를 예측하게 해주는 앱입니다.
